# Asanaコネクター - セクションにタスクを追加する
このアクションは、Asanaのセクションにタスクを追加します。

## 入力

| フィールド | 説明 |
|:--- |:--- |
| セクションID<br>**必須** | タスクを追加するセクションです。 |
| タスクID<br>**必須** | 対応するタスクを追加するためのタスクIDです。 |
| タスクIDの後に挿入 | このタスクを追加する前のタスクのIDです。 |
| タスクIDの前に挿入 | このタスクを追加する後のタスクのIDです。 |
{: .sdk-ref :}

## 出力

このアクションには出力はありません。
